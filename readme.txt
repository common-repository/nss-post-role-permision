== Descriptions ==

This Plugin is uesd to any body who is create a new user to access to read and insert post, pages and custom post. It will be custom post in you page, or a Template page. 

Cool Features:
*Just put short-code in your page or a Template. (short-code is: show_new_role) 
*custom role permission, access only read, insert
*Show custom post which is role is activated.
